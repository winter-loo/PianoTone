# tone-piano

An upgrade and fix for the repo [tambien/Piano](https://github.com/tambien/Piano)

- Use `pnpm` and `vite`
- Handle audio samples loading error
- A pretty demo page

To install dependencies:

```bash
pnpm install
```

To run dev server:

```bash
pnpm dev
```

To build library:

```bash
pnpm build:lib
```
